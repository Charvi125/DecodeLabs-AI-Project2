#  Rule-Based AI Chatbot

## DecodeLabs Artificial Intelligence Internship - Project 1

This project is a simple Rule-Based AI Chatbot developed using Python.

The chatbot responds to predefined user inputs using if-else statements and runs continuously until the user exits.

---

## Features

- Greeting responses
- AI-related questions
- Continuous conversation loop
- Exit command
- Beginner-friendly code

- VS Code
